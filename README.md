Toda vez que clonar o projeto e necessario criar um virtual enviroment e ativa-lo
Criacao do virtaul eviroment 
https://fastapi.tiangolo.com/virtual-environments/#create-a-project

Ativar o virtual enviroment 

```bash
.venv\Scripts\Activate.ps1
```


Para rodar o projeto voce instala a biblioteca do FastAPI 

```bash
pip install fastapi[standard]
```
___
pra rodar o comando voce usa 

```bash
fastapi dev main.py
```
